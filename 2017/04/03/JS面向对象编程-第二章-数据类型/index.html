发表于 2017-04-03 |

第二章 基本数据类型、数组、循环及表达式

  • 基本数据类型
    1.数字—-浮点数，整数：1、10000、3.23
    2.字符串—-“papa”,’123dfs’,’sasdf231243’
    3.布尔值—–true 和 false
    4.undefined—-不存在的变量，声明未赋值，undefined的值只有一个—undefined
    5.null—–包含一个值 没有值或空值
    剩下都是对象

    查看类型操作符—typeof

    typeof 123;
    “number”

var n=0x00;//0x开头，十六进制
typeof n;
“number”

Infinity // 超出JS处理范围
