backbone in underscore.js

发表于 2017-04-11 | 分类于 学习 |

backbone必须依赖underscore.js才能够使用，必须通过underscore.js的函数来完成访问页面元素等基本操作

backbone可以很好的与其他js库一起工作，所以说它是一个库，而不是框架

underscore调用_()方法进行封装，可以通过underscore的value()方法获取原生js对象中的数据

underscore template()函数声明

  1. <% %>逻辑代码，渲染后不会展现
  2. <%= %>数据类型，选然后展示数据
  3. <%- %>数据类型，将html转换常用字符串，避免XSS攻击

_.template(tmplString,[data],[setting])

1.underscore对象封装

var jsData = {
name:’data’
}

var underscoreData = _(jsData);

underscoreData.value();
//通过value方法获取原生数据

2.优先调用js1.6内置方法

3.改变命名空间

_下划线容易引起冲突，用noConflict()改变underscore的命名

var us = _.noConflict();

4.链式操作

jQuery中的链式操作

$(‘a’).css(‘position’,’relative’).attr(‘href’,’#’).show();
underscore同样支持链式操作，但要先声明chain()方法
var arr=[10,20,30];
_(arr).chain().map(function(item){return item++;}).first().value();
将调用的方法封装在一个闭包内，返回值封装为一个对象并返回

var result =function(obj,chain){

return chain?_(obj).chain():obj;

}

5.拓展underscore
通过mixin()方法向us中拓展自定义方法

_.mixin({

method1: function(object){
//todo
}
…
});

新方法追加到us原型对象中，所有创建的us对象都可以使用这些新方法

6.遍历集合
each(),map()迭代一个集合，并处理集合中的每一个元素

var arr = [1,2,3];

_(arr).map(function(item,i){

arr[i] = item + 1;

});

var obj ={
first:1,second:2
}
_(obj).each(function(value,key){
return obj[key] = value + 1;
});

map方法将返回的结果记录到一个新的数组并返回

7.函数节流
debounce() , throttle()
input提示内容
控制函数不被频繁调用，节省客户端及服务器资源
debounce 关注函数执行间隔
throttle 关注函数执行频率
